# Contributing

Thank you for your interest in contributing to Fumohouse! Contributions are
welcome to all Fumohouse projects. To get started:

1. Read the [guidelines common to all contributions](./common-guidelines.md).
1. For contributions containing code and/or scenes, read the [code contribution
   guidelines](./code-guidelines.md). For contributions containing assets, read
   the [asset contribution guidelines](./assets-guidelines.md). For other
   contributions, please reach out to us for more information.

If you have any questions, feel free to reach out to a maintainer.
